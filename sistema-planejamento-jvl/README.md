# Sistema de Monitoramento Estratégico — SED Joinville

Sistema de monitoramento de metas, OKRs e indicadores do Plano Estratégico Institucional (PEI) 2025-2029 da Secretaria Municipal de Educação de Joinville/SC.

## Tecnologias

- Google Apps Script
- Google Sheets (banco de dados)
- HTML5 / CSS3 / JavaScript
- Chart.js / D3.js

## Estrutura

- **Code.js** — Roteamento, configuração e lógica principal
- **DashboardService.js** — Agregação de dados para dashboards
- **DataService.js** — Operações de dados
- **Auth.js** — Autenticação e perfis (GERENCIA, PLANEJAMENTO, SECRETARIO)
- **App.html** / **Login.html** — Interface web

## Funcionalidades

- Acompanhamento de Objetivos Estratégicos (OE), Resultados, OKRs e Atividades
- Indicadores por eixo (Alunos, Professores, Gestão Escolar, Capacidade Institucional)
- Importação de dados da planilha de acompanhamento 2026
- Dashboards com métricas de execução e atingimento de metas
