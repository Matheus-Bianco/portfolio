Dashboard Executivo

Visão geral do PEI 2025-2029

Nota sobre os dados deste Dashboard
Os dados são extraídos automaticamente da planilha do Planejamento Estratégico 2026, com atualização automática diária às 7h, 12h e 17h. O percentual de execução é calculado com base no status das atividades: Concluída = 100%, demais = 0%. Atividades sem a coluna "Realizado" preenchida na planilha aparecem como "Sem Status".

Cronograma de Atividades 2026

Sobre o Cronograma: Este gráfico Gantt mostra a distribuição temporal de todas as atividades do PEI ao longo de 2026. As barras coloridas indicam o período planejado (início a término) de cada atividade. As cores refletem o status atual: verde = concluído, azul = em andamento/no prazo, vermelho = atrasado, laranja = replanejado, cinza = não iniciada. Use os filtros para focar em diretorias, OEs ou trimestres específicos.
